# 字幕拼接工具

<img src="assets/logo.png" alt="LOGO" style="zoom:25%;" />

一个纯前端的电影截图字幕拼接网页工具。可以导入多张截图，将首张图片与后续截图中的字幕区域拼接成一张长图，并支持水印和导出设置。

## 功能

- 多张图片导入，支持拖拽导入
- 下方时间轴预览，支持排序和删除
- 首张图片可单独设置下方裁剪位置，默认保留 100%
- 后续图片可设置字幕区域的上下边界
- 支持纵向或横向拼接
- 支持设置片段间距
- 支持水印文字、字体、字号、RGBA 颜色、位置和偏移
- 支持添加本机已安装字体名称作为自定义水印字体
- 支持自定义输出宽高、格式和质量
- 纯浏览器本地处理，不上传图片

## 使用方式

直接用浏览器打开 `index.html` 即可使用。

```text
index.html
styles.css
script.js
```

也可以使用任意静态文件服务器预览，例如：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 基本流程

1. 点击“导入图片”或将图片拖入预览区域。
2. 在下方时间轴中调整图片顺序。
3. 设置首张图片的下边界裁剪位置。
4. 设置后续图片的字幕区域上下边界。
5. 根据需要设置拼接方向、间距、输出尺寸和格式。
6. 如需水印，勾选“启用”并调整水印参数。
7. 点击“导出图片”保存结果。

## 自定义字体

水印字体支持输入本机已安装的字体名称。添加后会写入浏览器本地存储，刷新页面后仍会保留。

注意：浏览器只能使用当前电脑已经安装、且字体名称输入正确的字体。

## 技术说明

- 无后端服务
- 无构建步骤
- 使用 HTML、CSS、JavaScript 和 Canvas 实现
- 图片处理全部在浏览器本地完成
